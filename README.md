***Devanshu Chheda — Portfolio***

Personal developer portfolio built with vanilla HTML, CSS, and JavaScript. Features a dark purplish-blue theme, animated skill bars, project showcase, experience timeline, and a working contact form via Netlify Forms.

**Live Demo**
*devanshu-chheda-portfolio.netlify.app*

**Project Structure**

portfolio-final/
├── index.html            # Hero / About section
├── skills.html           # Tech skills with animated progress bars
├── projects.html         # Project cards with modal detail view
├── experience.html       # Experience & activities timeline
├── education.html        # Education cards
├── certifications.html   # Certifications grid
├── contact.html          # Contact details, social links & form
├── css/
│   ├── tokens.css        # Design tokens, colour palette, base reset
│   ├── layout.css        # Navbar, hero, containers, footer
│   └── components.css    # Buttons, cards, skill bars, modal, form
└── js/
    ├── theme.js          # Dark / light mode toggle
    ├── ui.js             # Typed text, photo upload, counters, toast
    └── projects.js       # Project modal logic

**Features**
  - Multi-page responsive layout — optimised for mobile, tablet, and desktop
  - Dark purplish-blue & black theme with light mode toggle
  - Animated skill bars with percentage achievement indicators
  - Project cards with modal pop-up for full details
  - Experience timeline with role, organisation, and skill tags
  - Education & Certifications sections
  - Working contact form powered by Netlify Forms (no backend required)
  - Responsive social links — LinkedIn & GitHub profiles
  - Scroll-reveal animations throughout

**Tech Stack - Getting Started**
- Run Locally
  - Clone the repository:
      git clone https://github.com/Devanshuhc29/portfolio-final.git
  - Open the project in VS Code
  - Install the Live Server extension
  - Right-click index.html → Open with Live Server
  - Deploy to Netlify
    - Drag and drop the portfolio-final/ folder to app.netlify.com/drop — no configuration needed.
